# 微信数据库解密与聊天记录搜索工具 v2.0

基于 WeChatDataAnalysis 项目重构，集成了数据库解密、聊天记录导出和搜索功能的 Web 应用。

## 🆕 v2.0 新功能

- **个股K线复盘** - 输入股票代码，生成带聊天标注的K线图
- **悬停显示聊天** - 鼠标悬停K线时显示当天相关聊天记录
- **智能搜索关联** - 同时搜索股票名称和代码

## 功能特性

- **数据库解密** - 支持首次完整解密和智能增量解密
- **聊天记录导出** - 解析并导出聊天记录到 Excel
- **智能搜索** - 按时间、聊天对象、发言人、关键词搜索
- **股票筛选** - A股/港股/美股名称匹配过滤
- **🆕 K线复盘** - 个股K线图与聊天记录关联分析
- **图片解密** - 自动提取密钥并解密微信缓存的.dat图片文件（仅V4格式）
- **AI问答** - 支持OpenAI兼容格式的大模型对话
- **配置管理** - 保存数据库密钥和路径配置

## 快速开始

### 方式一：使用EXE（推荐）

1. 双击 `国海金工微信数据筛选工具v2.0.exe`
2. 浏览器会自动打开并访问 http://127.0.0.1:5000
3. 如未自动打开，请手动访问上述地址

### 方式二：从源码运行

**安装依赖:**

```bash
# 使用 UV (推荐)
uv sync

# 或使用 pip
pip install flask flask-cors pandas openpyxl cryptography zstandard pillow pymysql baostock pyecharts
```

**启动服务:**

```bash
python web_app.py
```

## 使用指南

### 模块A：数据库解密与配置

1. 输入64位十六进制解密密钥
2. 输入数据库路径 (db_storage 目录)
3. 点击"解密数据库"

### 模块B：搜索聊天记录

1. 设置日期范围、聊天对象、关键词等条件
2. 可选择股票市场筛选（A股/港股/美股）
3. 点击"搜索聊天"
4. 支持导出搜索结果为Excel

### 模块C：AI分析系统

1. 配置API地址、模型名称、API密钥
2. 点击"测试连接"验证配置
3. 可导入搜索结果作为上下文进行AI对话
4. 🆕 按提示词逐条注入聊天记录，避免一次性加载过多文本

### 🆕 模块D：个股K线复盘

1. 输入股票代码（如 600519、000001）
2. 选择日期范围
3. 点击"生成K线复盘"
4. 查看K线图，鼠标悬停可见当天聊天记录
5. 下方显示相关聊天记录列表

**支持的股票代码格式:**
- A股：600xxx（上海）、000xxx/002xxx/300xxx（深圳）
- 示例：600519（贵州茅台）、000001（平安银行）

**注意:** K线数据来自baostock免费接口，部分新股或ETF可能无数据

## 技术细节

### 加密算法

- 密钥派生: PBKDF2-SHA512, 256,000 轮迭代
- 加密算法: AES-256-CBC
- 完整性验证: HMAC-SHA512
- 页面大小: 4096 字节

### K线复盘

- 数据源: baostock (免费A股数据)
- 图表库: pyecharts (交互式K线图)
- 渲染方式: iframe隔离渲染

### 消息类型

- 1: 文本
- 3: 图片
- 34: 语音
- 43: 视频
- 47: 表情
- 49: 链接/公众号
- 10000: 系统消息

## 打包为 EXE

```bash
# Windows: 双击运行
build.bat
```

打包完成后：
- 发布包位置: `dist_release/`
- 主程序: `dist_release/国海金工微信数据筛选工具v2.0.exe`
- 可将整个 `dist_release/` 文件夹打包为 zip 分发

**注意：**
- exe 文件约 600MB（包含 Python 运行时和数据依赖）
- 首次启动需要几秒钟解压临时文件
- config.xlsx 和 output/ 会在运行时自动创建
- 关闭控制台窗口会停止服务
